Multi-image Carousel

This Carousel will automatically display multiple slides at a time, dependent on the value you place in the multiSlides() parameter, located in the JavaScript code. (Example: multiSlides(3) to show three images).


Multi Image HTML Code

Keeping the id in the correct order is necessary for the carousel to function properly.
