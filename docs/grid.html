HTML Code

It is recommended to wrap your grid inside of a .container class

                    



<div class="container">

                    
    <div class="row">

        <div class="col-md-12 col--outline">12</div>

    </div>

                    
</div>

CSS Code for Boxes

                    

.col--outline {

    border: 1px solid #000063;

    border-radius: 3px;

    padding: 6px;

    text-align: center;

    font-weight: bold;

    margin-bottom: 5px;

}

It is also suggested to wrap your columns in a .row element. Here is an example of how these columns automatically adjust to the window size.
